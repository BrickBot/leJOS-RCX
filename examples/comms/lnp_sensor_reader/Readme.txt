This example uses the LNP Integrity layer protocol.
Run LNPReadSensor on the PC. It works either with the lejos
program LNPSensorReader or the legOS program lnptest.c.

To run lnptest.c you will need to install legOS and an H8300
cross-complier. See http://legOS.sourceforge.net

You need to start the lejos or legOS program before running the 
PC program.
