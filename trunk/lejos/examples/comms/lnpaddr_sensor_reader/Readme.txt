This example uses the LNP Addressing layer protocol.
Run LNPAddrReadSensor on the PC. It works either with the lejos
program LNPAddrSensorReader or the legOS program lnpaddrtest.c.

To run lnpaddrtest.c you will need to install legOS and an H8300
cross-complier. See http://legOS.sourceforge.net

You need to start the lejos or legOS program before running the 
PC program.

The LNP Addressing protocol supprts point to point connections and
communication between multiple RCXs.
